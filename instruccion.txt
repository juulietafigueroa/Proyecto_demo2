1) CREAR UN ENTORNO VIRTUAL, ESTO SE HACE UNA UNICA VEZ 

python3 -m venv NOMBREDELENTORNO

python3 -m venv django


2) ACTIVAR EL ENTORNO, CAPAZ QUE HAGA FALA HACERLO CUANDO ENTRAMOS AL VISUAL STUDIO

.\django\Scripts\Activate.ps1

Tenemos que ver adelante de la ruta que diga el nombre del ENTORNO como por ejemplo "(django)" entre parentesis:

(django) PS C:\Users\Juli\Documents\visual_studio_code\Django_env> 



3) instalamos las cosas una sola vez, ejemplo django:

pip3 install django


4) ahora supongamos querer ejecutar un programita

Comandos que usar para crear y conectar repositorio de git hub:
1. git init
2. git add . (agregar todos los archivos que queremos subir)
3. git commit -m 'nombre del commit'
4. git remote add origin (ruta)
5.git push origin main o master (primero probar con master y si tira error probar con main)
